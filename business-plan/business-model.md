# Red-Link Group 事業計画

更新: 2026-05-25

## ポジショニング

> 鉄緑会出身の医学部生による、オンライン医学部受験指導。

- **Phase 1（今）**: 在日中国人家庭から始める。WeChatで集客しやすく、競合ゼロ
- **Phase 2（10家庭以降）**: 日本人家庭にも展開。医学部実績を武器に市場を広げる

---

## サービス（2つ）

### ① オンライン個別指導（月89,800円）

高単価・高サポート。成果事例を作るための主力。

| 内容 | 詳細 |
|------|------|
| 授業 | 週1回×60分（月4回）、Zoom |
| 科目 | 数学・英語・理科（医学部受験の主要科目） |
| 書画カメラ | 手元の計算・記述をリアルタイムで確認 |
| 進路相談 | 月1回×30分（Phase 1は中国語、Phase 2は日本語） |
| 報告 | 月次レポート（Phase 1は中国語、Phase 2は日本語） |
| 初回 | 医学部合格ロードマップを面談後24時間以内に送付 |

追加オプション: 週2回 → +40,000円 / 2科目目 → +20,000円

---

### ② 単元別 少人数グループ指導（月29,800円）

スケールしやすい・入りやすい価格帯。個別指導への導線にもなる。

| 内容 | 詳細 |
|------|------|
| 授業 | 週1回×90分（月4回）、Zoom |
| 人数 | 4〜6人 |
| 形式 | 単元ごとに区切った短期集中型（例：有機化学・微積分・英文読解） |
| 言語 | 授業は日本語。質問・フォローは中国語対応可（Phase 1） |
| 書画カメラ | 手元を映して板書確認 |

**単元の例**

| 科目 | 単元例 |
|------|--------|
| 数学 | 微積分・確率・整数・ベクトル |
| 化学 | 有機化学・理論化学・計算問題 |
| 英語 | 英文読解・英作文・語彙 |

個別指導との違い：自分の苦手単元だけ受講できる。複数単元の受講も可。

## Phase 1：在日中国人ビーチヘッド

**なぜ中国人から始めるか**

- WeChatコミュニティで一気にリーチできる
- 医師は中国文化圏で最上位の職業 → 支払い意欲が高い
- 中国語対応 × 医学部実績の組み合わせは競合ゼロ

**集客**

| チャネル | 方針 |
|---------|------|
| WeChat | 知人ネットワーク・在日中国人グループ |
| SSTフェス（5/29-31） | 対面ブース、その場でWeChat追加 |
| Rednote | 医学部受験情報コンテンツ（週2〜3投稿） |

**クロージング**

- 無料面談（30分、中国語）→ 医学部合格ロードマップを翌日送付
- 面談その場で決めた方に**書画カメラ（1万円相当）をプレゼント**

---

## Phase 2：日本人家庭への拡大（10家庭達成後）

- 進路相談・レポートを日本語に切り替えるだけで同じサービスを提供できる
- Phase 1の合格実績が日本人向けの信頼証明になる
- 集客追加: Twitter/X・note（受験体験・勉強法）
