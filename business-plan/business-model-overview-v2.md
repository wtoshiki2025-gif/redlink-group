## 結論

最も良さそうなモデルは、**在日中国人家庭向けのオンライン受験ナビゲーション塾**。

単なるオンライン家庭教師ではなく、親向けの中国語進路相談、生徒向けのオンライン授業、学年・目的別のオンライン集団指導、そして**講師ネットワークとAI自習サポートへのアクセス権を提供する会員制コミュニティ**を組み合わせる。

事業の主力は、将来的には**オンライン集団指導**に置くべき。理由は、個別指導よりも1人あたり価格は下げられる一方、1授業あたりの売上は大きくできるからである。

ただし、初期から集団指導だけで売るのは難しい。最初は中国語進路相談で信頼を作り、少人数の個別指導で成果事例を作り、その後に学年別・目的別の集団指導へ広げるのが現実的である。

初期の収益目標は**月35万円から50万円**とし、全商品が理想通りに立ち上がった場合の上振れとして月80万円を見る。

## 市場規模の試算

計画の前提として、到達可能な市場の規模を把握しておく。

| 項目 | 数値 | 出典・根拠 |
| --- | ---: | --- |
| 在日中国人人口 | 約82万人 | 出入国在留管理庁統計 |
| うち永住・定住・家族滞在等の定着層 | 約50万人 | 在留資格別統計から推定 |
| うち子育て世帯（子ども5歳から18歳） | 約8万から10万世帯 | 世帯構成比から推定 |
| うち首都圏在住 | 約5万から6万世帯 | 地域分布から推定 |
| うち教育投資意欲が高い層（月3万円以上） | 約1万から2万世帯 | 中国人の教育投資傾向から推定 |
| うち難関校進学に関心がある層 | 約3,000から5,000世帯 | 上位層に絞った推定 |

Red-Link Groupが現実的に到達できる市場（SAM）は、首都圏の教育投資意欲の高い在日中国人世帯の中で、オンラインサービスに抵抗がない層。初期は**500から1,000世帯**がリーチ可能な上限と考える。

この市場規模から逆算すると、初期の顧客獲得目標は10から20家庭。中期で50から100家庭。これが計画全体の天井になる。

表面的な課題は「勉強を教えてほしい」だが、より深い課題は以下。

- 親が日本の受験制度を理解できず、正しい判断ができない
- 学校・塾から得た情報を親が咀嚼できない

- どの学年で何を準備すべきかが見えない

- 大手塾に入っても、親が学習状況を把握できない
- 集団塾の授業についていけないが、完全個別は高すぎる

## 商品設計

### 1. オンライン進路相談

親向けの商品。中国語で行う。最初の信頼獲得、課題整理、有料商品への導線として最重要。

### 商品ライン

| 商品 | 価格 | 内容 | 役割 |
| --- | ---: | --- | --- |
| 初回進路診断 | 0円から5,000円 | 45から60分相談、AI生成の簡易ロードマップ | 入口 |
| 単発進路相談 | 15,000円から30,000円 | 60から90分相談、資料確認、方針提案 | 収益化された入口 |
| 月次進路伴走 | 月30,000円から50,000円 | 月1回面談、月次レポート、学習計画修正 | 継続課金 |
| 受験直前伴走 | 月80,000円から120,000円 | 週次面談、志望校/併願/過去問管理 | 高単価短期 |

### 相談で扱うテーマ

- 日本の中学受験・高校受験・大学受験の仕組み
- 塾選び、転塾、併用の判断
- 志望校と現在地のギャップ
- 模試・定期試験・内申の見方
- 家庭学習計画
- 個別指導か集団指導かの判断

### 売り方

最初から「月額契約」を売るより、まずは初回診断でロードマップを見せる。相談後に以下の3択を提示する。

1. 進路相談だけ継続する
2. 集団指導に入る
3. 個別指導で弱点を潰す

この分岐設計にすると、相談が単なる無料対応で終わらず、次の商品に自然につながる。

### AI活用: 進路ロードマップの半自動生成

初回相談後に「簡易ロードマップを24時間以内に送付」するために、AIを活用する。

- ヒアリングシートの回答（学年・成績・志望校・現在の塾）をAIに入力
- 志望校別の合格に必要な偏差値・内申・科目配分のデータベースを参照し、子どもの現在地とのギャップをAIが自動算出
- ロードマップの草案をAIが生成し、講師が最終確認・調整を加える

これにより、ロードマップ作成が3時間から30分に短縮される。初回相談の付加価値を維持しつつ、対応家庭数を大幅に増やせる。

### 2. オンライン個別指導

生徒向けの商品。授業は原則日本語で行い、保護者への説明・報告は中国語で行う。

### 位置づけ

個別指導は、売上の初期立ち上げには強いが、創業者の時間を消費しやすい。そのため、長期的な主力というより、以下の用途で使う。

- 成果事例を作る
- 高単価家庭に深く入る
- 集団指導では補えない弱点を潰す
- 受験直前期の単価を上げる

### 商品ライン

| 商品 | 価格 | 内容 | 対象 |
| --- | ---: | --- | --- |
| 個別ライト | 月60,000円から80,000円 | 週1回60から90分、月次報告 | 苦手科目1つ |
| 個別スタンダード | 月100,000円から150,000円 | 週1から2回、学習計画、保護者面談 | 難関校志望 |
| 個別プレミアム | 月180,000円から250,000円 | 週2回以上、週次管理、直前期対応 | 高単価・直前期 |

### 価格の考え方

個別指導は「医学部生が教える時間」だけで価格を説明しない。以下を含める。

- 授業
- 宿題設計
- 学習ログ管理
- 保護者への中国語報告（AI草案 + 講師確認）
- 進路ロードマップ修正

これにより、一般的なオンライン家庭教師ではなく、進路伴走つき個別指導として価格を正当化できる。ただし、初期で実績がない段階では個別スタンダード（月10万円）から入る家庭は少ない。**最初の数家庭は個別ライト（月6万から8万円）で始め、成果事例ができてから価格を上げる**のが現実的。

### AI活用: 保護者向け中国語レポートの自動生成

- 授業後のメモ（「今日は二次関数。場合分けが弱い」等）をAIに渡すと、中国語の保護者向け月次レポートを生成
- 確認テストの結果を入力すると、AIが「次回までにやるべきこと」を中国語で保護者向けに記述
- 講師は最終確認のみ。1家庭あたりのレポート作成が30分から5分に短縮

### 3. オンライン集団指導

中期の主力商品。最も伸ばすべき領域。

### なぜ集団指導が重要か

個別指導は1家庭あたりの単価は高いが、講師の時間がボトルネックになる。集団指導は1人あたり価格を個別より下げても、1授業あたりの売上を大きくできる。

| 形式 | 受講者数 | 1人あたり月額 | 1クラス月商 |
| --- | ---: | ---: | ---: |
| 個別指導 | 1人 | 100,000円 | 100,000円 |
| 少人数集団 | 6人 | 40,000円 | 240,000円 |
| 集団講座 | 12人 | 25,000円 | 300,000円 |
| 短期講習 | 20人 | 15,000円 | 300,000円 |

同じ講師時間でも、集団指導の方が授業単価は高くなる。さらに、授業を録画して復習用コンテンツにすれば、コミュニティや欠席フォローにも転用できる。

### 集団指導の基本設計

| 人数 | 初期は4から8人、慣れたら10から15人 |

| 価格 | 月25,000円から50,000円 |

### 集団指導の現実的なハードル

「中1から中2」「数学先取り」「在日中国人」「オンライン」で同学年・同レベルの生徒を4人以上揃えるには、見込み客40から100家庭のプールが必要（転換率5から10%）。集団指導の開講は、**コミュニティと進路相談で見込み客が十分に溜まってから**にする。無理に初月から開講しない。

### 集団指導の売り方

集団指導は「安い個別指導」として売らない。以下のように売る。

- 同じ目標を持つ少人数クラス
- 難関受験経験者が設計したカリキュラム
- 授業後に確認テストと課題がある
- 親には中国語で進捗が見える
- 授業外でもAI自習サポートが使える
- 必要なら個別指導を追加できる

### 4. 会員制コミュニティ（資源アクセス権モデル）

コミュニティの価値は「情報アーカイブ」ではなく、**Red-Link Groupの資源に継続的にアクセスできる権利**である。会員である限り、講師ネットワーク・AI自習サポート・保護者同士のつながりを利用できる。退会するとこれらを失う。

この設計により、情報を消費し尽くして退会される構造を回避し、受験が終わるまで継続する理由を作る。

### コミュニティで提供するもの

| 提供するもの | 内容 |
| --- | --- |
| 講師ネットワークへのアクセス | 「こういう科目・学年に対応できる講師がいます」を知れる。分野別Q&Aで直接質問できる |
| AI自習サポート | 問題の写真を送ると解説が返る。ヒント段階的提示。授業外の自習を支える |
| 月1回の分野別Q&Aライブ | 数学・英語・受験制度等のテーマで講師がライブ回答 |
| 月1回の保護者向けロードマップ講座 | 中国語での受験制度解説・学年別戦略 |
| 保護者同士の情報交換 | 「この塾どうだった？」「この学校の説明会は？」等 |
| 過去セミナーアーカイブ | 録画・資料の閲覧 |
| 有料商品の優先案内・割引 | 進路相談・集団指導・個別指導の会員価格 |

### コミュニティで提供しないもの（有料商品に回すもの）

| 有料商品に回すもの | 理由 |
| --- | --- |
| 個別の成績分析・学習計画作成 | 進路相談（有料）の領域 |
| 実際の授業・個別指導 | 個別指導・集団指導（有料）の領域 |
| 志望校別の戦略面談 | 進路相談（有料）の領域 |
| 週次の学習管理・報告 | 個別指導（有料）の領域 |
| 受験直前の集中対策 | 短期講習（有料）の領域 |

この線引きにより、コミュニティは「味見の場」として機能し、本格的に使いたい家庭は有料商品に自然に移行する。

### コミュニティの商品設計

| プラン | 価格 | 内容 | 役割 |
| --- | ---: | --- | --- |
| 無料会員 | 0円 | 月1本の受験コラム、セミナー告知 | リード獲得 |
| スタンダード会員 | 月5,000円から8,000円 | Q&Aライブ、講師ネットワーク紹介、AI自習サポート、アーカイブ、保護者コミュニティ、有料商品の会員割引 | 資源アクセス権 |
| 既存生（個別/集団受講者） | 受講料に含む | スタンダード会員の全特典 | 継続率向上 |

旧案の3,000円から値上げし、5,000円から8,000円とする。理由は、AI自習サポートと講師ネットワークアクセスが加わることで、「塾の月謝の10分の1で受験の伴走者を得られる」という価値が成立するため。塾月謝5万円から10万円と比較すれば、月5,000円から8,000円は十分安い。

旧案のプレミアム（月9,800円から15,000円）は廃止する。中間プランが多すぎると選択疲労を起こし、かつ「プレミアムで足りるなら進路相談は不要」と思われ、有料商品への転換率が下がる。

### どう売るか

1. RED NOTE/WeChatで無料の受験情報を発信する
2. 無料オンラインセミナーに集める
3. セミナー最後に「講師への質問・AI自習サポート・毎月の講座はコミュニティへ」と案内する
4. コミュニティ内で個別相談枠を販売する
5. 個別相談から集団指導・個別指導へ転換する

初期は大きなプラットフォームを作らない。WeChatグループ、Notion/Google Drive、Zoomで十分。AI自習サポートもWeChatまたはLINE上のチャットボットで始める。

| --- | --- | --- |
| 1ヶ月目 | 日本の中学受験・高校受験の全体像 | 小4から中2の親 |
| 2ヶ月目 | 塾選びと転塾判断 | すでに塾に通う家庭 |
| 3ヶ月目 | 模試・成績表の読み方 | 小5から中3の親 |
| 4ヶ月目 | 内申と高校受験の仕組み | 中学生の親 |
| 5ヶ月目 | 難関校を目指す英数の勉強法 | 中1から高1 |
| 6ヶ月目 | 夏休みの学習計画 | 全学年 |

### コミュニティの運営負荷コントロール

コミュニティ内で個別相談を無制限に受けると、低単価なのに運営負荷が重くなる。以下のように制限する。

- Q&Aは月1回のライブでまとめて回答
- 個別成績表の読み込みは有料相談に回す
- チャットは運営連絡と簡単な質問に限定
- 個別の質問対応は「まずAI自習サポートに聞いてください」と誘導し、AIで解決しない場合のみQ&Aライブで取り上げる
- 個別指導レベルの質問対応は既存生プランに含める

## AI活用基盤

AI活用は「AIで授業する」ではなく、**「AIで創業者2人のボトルネックを緩和し、より多くの家庭に対応できるようにする」**ことが目的。

### AI活用の全体像

| 優先度 | 活用領域 | 効果 | 必要な準備 |
| --- | --- | --- | --- |
| 1 | 進路ロードマップの半自動生成 | 対応家庭数を3倍に | 志望校別データベース構築 |
| 2 | 保護者向け中国語レポートの自動生成 | レポート作成を30分から5分に | レポートテンプレート設計 |
| 3 | AI自習サポート（会員向け） | コミュニティの退会理由を潰し、講師の質問対応負荷を下げる。他塾にない差別化 | チャットボット構築 |
| 4 | 中国語コンテンツの量産 | RED NOTE/WeChat投稿を週3で継続可能に | プロンプト設計 |
| 5 | 見込み客の自動ナーチャリング | フォロー漏れ防止 | CRM連携（100家庭超えてから） |

### 1. 進路ロードマップの半自動生成（優先度1）

初回相談後にロードマップを送るプロセスを効率化する。

**入力:** ヒアリングシート回答（学年、現在の成績、志望校、現在の塾、親の悩み）

**AI処理:**
- 志望校別の合格に必要な偏差値・内申・科目配分を参照
- 子どもの現在地とのギャップを算出
- 学年ごとの学習計画・塾選び・模試スケジュールを含むロードマップ草案を生成
- 中国語で出力

**講師の作業:** 内容の確認・調整（15から30分）

**効果:** ロードマップ作成が3時間から30分に。月10家庭の相談でも対応可能。

### 2. 保護者向け中国語レポート自動生成（優先度2）

授業後の保護者報告を効率化する。

**入力:** 授業メモ（「二次関数。場合分けが弱い。宿題は問題集p.45-48」等）

**AI処理:**
- 保護者向けの中国語レポートを生成（今日の授業内容、子どもの理解度、次回までにやるべきこと）
- 月次の到達度サマリーを自動集計

**講師の作業:** 最終確認（5分/家庭）

**効果:** 月次レポート作成が1家庭30分から5分に。20家庭でも2時間で完了。

### 3. AI自習サポート（優先度3）

コミュニティ会員・既存生向けに、授業外の自習をAIで支える。

**機能:**
- 問題の写真を送ると、AIが解説を生成（日本語）
- いきなり答えを出さず、ヒントを段階的に提示する設計
- 「この問題がわからない」→ AIが関連する基礎概念を提示 → それでもわからなければ解法を説明
- AIで解決しない場合は「次のQ&Aライブで講師に聞きましょう」と導線を作る

**技術的な実装:**
- 初期はWeChat/LINE上のチャットボット（Claude API等を利用）
- 数学・理科は画像認識対応
- 対応範囲は中学数学・英語から開始し、需要に応じて拡大

**効果:**
- 「授業以外の日もサポートがある」という価値がコミュニティに追加される
- 退会するとAI自習サポートも失うため、継続率が上がる
- 講師への質問が減り、運営負荷が下がる
- **他の在日中国人向け塾にはない差別化要素**になる

**注意:** AIの回答精度は完璧ではない。「AIが出した解説を鵜呑みにしない」「わからなければ講師に聞く」という利用ガイドラインを設ける。

### 4. 中国語コンテンツの量産（優先度4）

RED NOTE・WeChatへの投稿を週3回で継続するために活用する。

**プロセス:**
- 日本語の受験情報ソース（塾比較サイト、入試要項、模試情報等）をAIで中国語に変換
- RED NOTE向けのフォーマット（短文+画像+ハッシュタグ）に整形
- 「今週の受験ニュース」を自動要約
- 最終チェックのみ人間が行う

**効果:** 週3投稿を1人で30分/日以内に回せる。

### 5. 見込み客の自動ナーチャリング（優先度5）

見込み客が100家庭を超えてから導入する。初期は手動で十分。

**機能:**
- セミナー参加者に自動フォローアップ（WeChat）
- 子どもの学年に応じて最適なコンテンツを自動配信
- 「小6の親にはこの記事」「中2の親にはこの記事」をAIが選別

## 集客設計

### 最初の10家庭をどう集めるか

計画全体で最も重要なのは、最初の見込み客を確保すること。具体的な行動を以下に定める。

| 順序 | アクション | 目標 | 時期 |
| --- | --- | --- | --- |
| 1 | 自分たちの知人・友人の在日中国人ネットワークにサービスを伝える | 5家庭にリーチ | 即日 |
| 2 | WeChat既存グループで受験情報を発信し始める | 20人の保護者に認知 | 1週間以内 |
| 3 | 無料オンラインセミナー第1回を開催 | 10家庭参加 | 2週間以内 |
| 4 | Co-Createの初期ユーザーテスト（5/16） | 5家庭と対面相談 | 5/16 |
| 5 | セミナー参加者・テスト参加者にコミュニティ案内 | 5家庭が会員登録 | 3週間以内 |

### 優先チャネル

1. WeChatの知人・在日中国人コミュニティ（最速）
2. 中国語の無料オンラインセミナー（信頼構築）
3. 小红书/RED NOTEでの受験情報発信（中長期の集客基盤）
4. 既存顧客からの紹介（最も効率的）
5. Co-Create等の対面相談会（初期のみ）
6. 中国人学校・補習校・華人団体への接触（要検討）

### RED NOTE方針

`docs/rednote_knowledge.md` の内容から、RED NOTEは広告色を抑えた情報発信が向いている。初期は以下の投稿テーマに絞る。

- 在日中国人保護者が知らない日本の受験制度
- 小5から中3でやるべき受験準備
- 難関中高・大学受験のロードマップ
- 塾選びで失敗しやすいポイント
- 親が日本語で困る学校・塾コミュニケーション例
- 日本の模試・偏差値・内申の見方

外部リンク、QRコード、WeChat IDの直接掲載は避ける。投稿では売り込まず、無料セミナーやプロフィール導線に自然に流す。

AIでコンテンツを量産する際も、「日本語の受験情報を機械翻訳しただけ」にならないよう、自分たちの経験・視点を必ず入れる。

## 初期検証設計

既存のCo-Create計画を活かし、検証対象を明確にする。

### 検証したい仮説

| 仮説 | 判断基準 |
| --- | --- |
| 中国語での進路説明に強い価値を感じる | 5名中3名以上が「必要」と回答 |
| 進路ロードマップが購買意欲を上げる | 5名中3名以上が「自分の子ども版が欲しい」と回答 |
| 月3万円以上の進路相談に支払い余地がある | 5名中2名以上が検討 |
| 月4万円前後の少人数集団指導に支払い余地がある | 5名中2名以上が検討 |
| 月6万から8万円の個別指導（ライトプラン）に支払い余地がある | 5名中1名以上が検討 |
| 月5,000から8,000円のコミュニティ（講師アクセス+AI自習サポート付き）に支払い余地がある | 5名中3名以上が検討 |
| AI自習サポートに魅力を感じる | 5名中3名以上が「子どもに使わせたい」と回答 |
| 相談後に次の行動が起きる | 連絡先提供、紹介、次回面談希望のいずれかが3件以上 |

### ユーザーテストで必ず聞くこと（追加項目）

価格感度だけでなく、以下も必ず確認する。

- **現在の教育費の実額**: 今、塾・家庭教師・教材に月いくら払っているか
- **現在のサービスへの不満**: 何が足りないか、何に困っているか
- **情報収集方法**: 受験情報をどこで得ているか（WeChat、RED NOTE、知人、日本語サイト等）
- **AI自習サポートへの反応**: 「子どもが自習中に問題の写真を送ると解説が返るサービスがあったら使うか？」
- **コミュニティへの反応**: 「月5,000から8,000円で講師に質問でき、AI自習サポートも使え、保護者同士で情報交換できるグループがあったら入るか？」

### ヒアリングで見るべき行動

- その場で家族や友人に紹介しようとするか
- 具体的な子どもの成績・志望校を共有するか
- 次回相談の日程を取りたがるか
- 集団指導の開講時期を聞くか
- コミュニティの月額を聞いても関心が落ちないか
- 既存の塾や家庭教師と比較して語るか
- AI自習サポートについて子どもに使わせたいと具体的に言うか

「いいですね」は需要の証拠にしない。次の行動を重視する。
