# Red-Link Group サービス説明資料

## 在日中国人家庭向けオンライン進学支援

Red-Link Groupは、日本の受験制度に不安を持つ在日中国人家庭に向けて、オンラインで進路相談・学習指導・保護者向け情報提供を行うサービスです。

子どもへの授業は日本語で行い、保護者への説明・進路相談・学習報告は中国語で行うことを想定しています。

## 解決したい課題

- 日本の受験制度が複雑で、親が全体像をつかみにくい
- 塾や学校からの説明を理解しきれず、家庭内で判断しづらい
- 志望校までに何をすべきかが見えない
- 子どもの学習状況を親が把握しにくい
- 中国語で信頼して相談できる進路支援先が少ない

## 商品1: オンライン進路相談

保護者向けの中国語相談サービスです。

| 内容 | 詳細 |
| --- | --- |
| 対象 | 受験制度、塾選び、志望校、学習計画に不安がある保護者 |
| 形式 | オンライン面談 |
| 提供物 | 簡易進路ロードマップ、学習方針、次にやること |
| 価格目安 | 単発15,000円から30,000円 / 月額30,000円から50,000円 |

## 商品2: オンライン個別指導

生徒向けの個別指導です。授業に加えて、保護者向けの進捗報告を行います。

| 内容 | 詳細 |
| --- | --- |
| 対象 | 苦手科目を集中的に伸ばしたい生徒 |
| 形式 | 週1回から2回のオンライン授業 |
| 科目 | 数学、英語、理科を中心に想定 |
| 価格目安 | 月60,000円から150,000円 |

## 商品3: オンライン集団指導

同じ目標・学力帯の生徒を集めた少人数オンライン授業です。

| 内容 | 詳細 |
| --- | --- |
| 対象 | 難関校を目指す小中高生 |
| 形式 | 週1回90分の少人数クラス |
| 特徴 | 確認テスト、宿題、保護者向けレポート |
| 価格目安 | 月35,000円から50,000円 |

## 商品4: 中国語受験情報コミュニティ

保護者が日本の受験情報を継続的に学べる会員制サービスです。

| 内容 | 詳細 |
| --- | --- |
| 対象 | まず情報収集から始めたい保護者 |
| 提供物 | 月1回講座、資料、Q&A、アーカイブ |
| 役割 | 進路相談や授業に入る前の情報整理 |
| 価格目安 | 月2,980円から5,000円 |

## Red-Link Groupの特徴

- 日本の難関受験を経験した医学部生・難関受験経験者が設計
- 子どもには日本語で授業、親には中国語で説明
- 授業だけでなく、進路ロードマップと学習管理まで支援
- 個別指導だけでなく、少人数集団指導と保護者向け情報提供を組み合わせる

## 最初の利用イメージ

1. 初回進路診断で現状と目標を整理する
2. 子ども専用の簡易ロードマップを作る
3. 必要に応じて、進路相談・集団指導・個別指導を選ぶ
4. 保護者は月次レポートや講座で学習状況を把握する
