# 分析フレームワーク

## 基本方針

日本人モニターの結果を、そのまま在日中国人家庭の需要として扱わない。結果は以下の3分類に分ける。

1. 普遍的価値
2. ターゲット特化価値
3. 再検討対象

## 1. 普遍的価値

日本人保護者でも強い反応が出た要素。これは、在日中国人家庭でも価値になり得る土台として扱う。

例:

- 進路ロードマップが欲しい
- 模試や内申の読み方を教えてほしい
- 塾選びを相談したい
- 集団指導でも確認テストと宿題管理があれば価値を感じる
- オンライン授業でも、授業後レポートや確認テストがあれば不安が下がる
- 受験情報を毎月整理して受け取りたい

判断:

- 5名中3名以上が強い関心を示したら、商品設計に残す
- 価格が課題でも、内容価値が強ければ価格・見せ方を調整する

## 2. ターゲット特化価値

日本人保護者では反応が中程度でも、「日本語で情報を取りにくい家庭なら価値が上がる」と評価された要素。

例:

- 中国語での保護者面談
- 中国語での月次レポート
- 中国語での受験制度解説
- 日本の塾・学校との情報ギャップを埋める相談

判断:

- 日本人モニターの直接支払意欲では判断しない
- 5名中3名以上が「言語差がある家庭なら価値が上がる」と答えれば、次回の中国人家庭ヒアリングで重点検証する

## 3. 再検討対象

日本人保護者にも伝わらず、在日中国人家庭に置き換えても価値が上がりにくい要素。

例:

- 商品が多すぎて違いがわからない
- コミュニティが単なるチャットグループに見える
- AI活用が価値ではなく不安として受け取られる
- 集団指導が普通のオンライン塾と区別できない

判断:

- 2名以上が同じ違和感を出したら、説明資料を修正する
- 3名以上が不要と答えたら、初期商品から外すか優先度を下げる

## 商品別の分析観点

### オンライン進路相談

見るべきこと:

- 相談だけで払うのか、ロードマップ込みなら払うのか
- 単発と月次のどちらが自然か
- 相談内容が塾の面談とどう違うと感じられるか

読み替え:

- 日本人でも価値が強ければ、普遍的な親向け顧問ニーズがある
- 日本人では弱いが、言語差がある家庭なら価値が上がる場合は、中国語対応を前面に出す

### オンライン個別指導

見るべきこと:

- 医学部生・難関受験経験者の価値が伝わるか
- 月6万から8万円の個別ライトが入口として現実的か
- 月10万円以上に必要な追加価値は何か
- 体験授業、先生プロフィール、先生変更制度が申し込み不安を下げるか
- 授業後レポートや保護者面談が、対面家庭教師との差別化になるか

読み替え:

- 日本人家庭が高いと感じる場合でも、中国語保護者報告込みで価格納得度が上がるかを見る

### オンライン集団指導

見るべきこと:

- 少人数集団の価値が伝わるか
- 何人までなら納得できるか
- 確認テスト、宿題管理、保護者レポートが価格を支えるか
- 集中力、質問しやすさ、学力差、競争感への不安がどれくらい強いか
- 録画視聴、欠席振替、進捗ダッシュボードが価格納得度を上げるか

読み替え:

- 日本人でも価値が強ければ、中期主力商品として検討しやすい
- 人数や学力差への不安が強ければ、最初は4人程度の小さなクラスで設計する

### オンライン授業の不安・必要機能

見るべきこと:

- 申し込みを止めるほど強い不安は何か
- 個別指導と集団指導で、不安の種類がどう変わるか
- 保護者が「必須」と感じる機能と「あればよい」機能は何か
- 対面授業と同価格でも納得できるか、オンラインなら安くないと難しいか

判断:

- 5名中3名以上が同じ不安を挙げたら、商品説明で先回りして解消する
- 5名中3名以上が高評価した機能は、初期提供範囲に入れる候補にする
- 価格納得の条件として出た機能は、価格表の内容欄に明記する

特に見る機能:

| 機能 | 読み方 |
| --- | --- |
| 体験授業 | 先生・授業形式への不安が強いサイン |
| 先生プロフィール | 講師品質の証明が必要なサイン |
| 先生変更制度 | 相性リスクを保護者が重く見ているサイン |
| 授業後レポート | 保護者への可視化が価値になるサイン |
| 確認テスト | 理解度保証へのニーズが強いサイン |
| 宿題管理 | 授業外管理まで期待されているサイン |
| 録画・アーカイブ | 欠席・復習・親の確認ニーズがあるサイン |
| 進捗ダッシュボード | 継続利用時の管理画面ニーズがあるサイン |

### 中国語受験情報コミュニティ

見るべきこと:

- 月額課金の理由が伝わるか
- 月1回講座・資料・Q&A・アーカイブが価値に見えるか
- チャットグループと誤解されていないか

読み替え:

- 日本人でも「資料と講座なら払う」となる場合は、情報商品として成立可能性がある
- 日本人には不要でも、言語差がある家庭なら価値が上がる場合は、中国語コンテンツの検証へ進む

## 結論の書き方

避けるべき表現:

- 日本人モニターでも好評だったので、在日中国人家庭にも売れる
- 中国語対応は価値があると証明された
- コミュニティの需要が検証できた

使うべき表現:

- 日本人モニターでも進路ロードマップの価値は一定確認できた
- 中国語対応は直接検証できていないが、第三者視点では価値が上がる可能性が示唆された
- コミュニティは、チャットではなく講座・資料・Q&Aとして見せる必要がある
