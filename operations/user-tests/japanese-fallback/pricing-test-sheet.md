# 価格感度確認シート

## 使い方

各商品について、以下の4つの価格を聞く。

1. 高すぎて検討しない価格
2. 高いが内容次第で検討できる価格
3. お得に感じる価格
4. 安すぎて品質が不安になる価格

日本人家庭としての感覚を先に聞き、その後で「日本語で受験情報を取りにくい在日中国人家庭ならどうか」を聞く。

## 商品別価格案

### 1. オンライン進路相談

| プラン | 価格案 | 内容 |
| --- | ---: | --- |
| 初回進路診断 | 0円から5,000円 | 45から60分相談、簡易ロードマップ |
| 単発進路相談 | 15,000円から30,000円 | 60から90分相談、資料確認、方針提案 |
| 月次進路伴走 | 月30,000円から50,000円 | 月1回面談、月次レポート、学習計画修正 |
| 受験直前伴走 | 月80,000円から120,000円 | 週次面談、志望校・過去問・併願管理 |

確認すること:

- 単発相談ならいくらまで払えるか
- 月額伴走なら何が含まれていれば納得できるか
- 相談だけで払うのか、ロードマップ込みなら払うのか

### 2. オンライン個別指導

| プラン | 価格案 | 内容 |
| --- | ---: | --- |
| 個別ライト | 月60,000円から80,000円 | 週1回60から90分、月次報告 |
| 個別スタンダード | 月100,000円から150,000円 | 週1から2回、学習計画、保護者面談 |
| 個別プレミアム | 月180,000円から250,000円 | 週2回以上、週次管理、直前期対応 |

確認すること:

- 一般的な家庭教師と比べて高いか安いか
- 保護者向けレポートや進路設計が価格に影響するか
- 医学部生・難関受験経験者という要素に価値を感じるか

### 3. オンライン集団指導

| プラン | 価格案 | 内容 |
| --- | ---: | --- |
| 少人数集団 | 月35,000円から50,000円 | 週1回90分、4から8人、確認テストあり |
| 難関受験演習 | 月50,000円から80,000円 | 週1から2回、過去問・弱点補強 |
| 短期講習 | 1講座15,000円から30,000円 | 季節講習、テーマ別講座 |

確認すること:

- 何人までなら少人数として納得できるか
- 個別指導ではなく集団指導を選ぶ理由があるか
- 授業後テスト、宿題管理、月次レポートで価格納得度が上がるか

### 4. 中国語受験情報コミュニティ

| プラン | 価格案 | 内容 |
| --- | ---: | --- |
| 無料会員 | 0円 | 受験コラム、セミナー告知 |
| ベーシック | 月2,980円から5,000円 | 月1回講座、資料、アーカイブ |
| プレミアム | 月9,800円から15,000円 | Q&A、限定資料、相談割引 |

確認すること:

- チャットグループではなく、資料と講座なら払うか
- 月1回の講座テーマとして何が欲しいか
- 個別相談割引があると継続しやすいか

## 在日中国人家庭に置き換えた質問

- 日本語で受験情報を取りにくい家庭なら、この価格の納得度は上がると思いますか。
- 中国語で保護者説明・月次レポートがある場合、どの商品の価値が一番上がりそうですか。
- 中国語対応があることで、追加で月いくら程度の価値がありそうに見えますか。
- 日本人家庭には不要だが、在日中国人家庭なら必要そうな支援は何ですか。
