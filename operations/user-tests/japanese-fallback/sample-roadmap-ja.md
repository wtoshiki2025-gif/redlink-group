# サンプル進路ロードマップ

## 想定ケース

| 項目 | 内容 |
| --- | --- |
| 学年 | 中学1年生 |
| 目標 | 首都圏の難関高校受験 |
| 現在の状況 | 学校成績は上位だが、受験塾には未通塾 |
| 得意科目 | 数学 |
| 課題科目 | 英語、国語の読解 |
| 保護者の悩み | いつから塾に入るべきか、内申と受験勉強の両立がわからない |

## 現在地の整理

### 強み

- 数学への抵抗が少なく、先取り学習に向いている
- 学校成績が安定しており、内申を取りやすい
- まだ中1のため、受験までの準備時間がある

### 課題

- 高校受験の全体像を家庭が把握できていない
- 英語の単語・文法・長文読解の積み上げが不足しやすい
- 国語読解は後回しになりやすく、伸ばすのに時間がかかる
- 模試を受けていないため、学校外での位置がわからない

## 6ヶ月以内にやること

| 領域 | やること | 目的 |
| --- | --- | --- |
| 受験情報 | 高校受験の仕組み、内申、模試、併願制度を理解する | 親の判断軸を作る |
| 数学 | 中1範囲を固め、中2範囲の先取りを始める | 難関校向けの土台作り |
| 英語 | 文法と単語を毎週管理する | 中2以降の長文対策に備える |
| 国語 | 読解問題を週1回入れる | 後回しリスクを避ける |
| 模試 | まず1回外部模試を受ける | 学校外での現在地を知る |

## 1年以内にやること

| 領域 | やること | 目的 |
| --- | --- | --- |
| 志望校 | 3段階で候補校を整理する | 目標校、現実校、安全校を見える化 |
| 数学 | 中2範囲を終え、関数・図形を強化する | 入試頻出単元に早く入る |
| 英語 | 長文読解と英作文を始める | 入試形式への接続 |
| 国語 | 説明文・物語文の解法を学ぶ | 点数の安定化 |
| 内申 | 定期テストと提出物を管理する | 受験校の選択肢を広げる |

## 中3前半にやること

| 領域 | やること | 目的 |
| --- | --- | --- |
| 志望校 | 第一志望と併願候補を仮決定する | 学習優先順位を決める |
| 数学 | 入試標準問題を一通り解く | 実戦力の土台作り |
| 英語 | 長文演習を毎週行う | 読む速度と精度を上げる |
| 国語 | 記述問題の添削を受ける | 独学で見えにくい弱点を潰す |
| 模試 | 2ヶ月に1回受ける | 志望校との差を追う |

## 中3後半にやること

| 領域 | やること | 目的 |
| --- | --- | --- |
| 過去問 | 志望校別に演習する | 出題傾向に慣れる |
| 弱点補強 | 模試と過去問から弱点を抽出する | 点数に直結する改善 |
| 併願 | 出願校を確定する | リスク管理 |
| 面談 | 月1回以上、学習方針を確認する | 家庭内の不安を減らす |

## Red-Link Groupで支援できること

| 課題 | 支援 |
| --- | --- |
| 親が受験制度を理解しきれない | オンライン進路相談 |
| 学習計画が立てられない | 進路ロードマップ作成 |
| 数学・英語を伸ばしたい | オンライン集団指導または個別指導 |
| 子どもの進捗が見えない | 月次レポート |
| 情報収集を継続したい | 受験情報コミュニティ |

## 保護者に確認したいこと

- このロードマップは、家庭の意思決定に使えそうか
- 自分の子ども用に作ってほしいと思うか
- どの情報が足りないか
- 有料で作るなら、いくらまでなら検討できるか
