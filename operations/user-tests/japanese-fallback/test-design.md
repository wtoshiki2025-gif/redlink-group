# テスト設計: 日本人モニター対応版

## 背景

Red-Link Groupの本来ターゲットは、在日中国人家庭の保護者である。特に、日本の受験制度や塾選びに不安があり、中国語で進路相談・学習状況説明を受けたい家庭を想定している。

今回の初期ユーザーテストでは、中国人モニターを調査会社が確保できなかったため、日本人保護者を代替モニターとして実施する。

## テスト目的

目的は、在日中国人家庭の需要を直接証明することではない。以下の4点を確認する。

1. 日本の受験支援サービスとして、進路相談・ロードマップ・学習管理に価値があるか
2. オンライン個別指導、オンライン集団指導、オンライン進路相談、コミュニティの違いが伝わるか
3. 価格帯が教育熱心な保護者にとってどの程度受け入れられるか
4. 日本人保護者から見て、言語・文化差がある家庭なら価値が上がりそうか

## モニター条件

優先条件は以下。

| 優先度 | 条件 | 希望 |
| --- | --- | --- |
| 1 | 子どもの学年 | 小5から中3 |
| 2 | 受験関心 | 中学受験、高校受験、難関校進学に関心あり |
| 3 | 塾利用経験 | 現在または過去に塾・家庭教師を利用 |
| 4 | 保護者の関与 | 塾選び、進路判断、学習管理に関与している |
| 5 | 世帯属性 | 教育投資に月3万円以上を検討できる |

小5から中3が難しい場合は、小4、高1、高2の保護者も可とする。ただし、分析時に学年差を分けて見る。

## 検証仮説

| 仮説 | 日本人モニターでの確認方法 | 判断基準 |
| --- | --- | --- |
| 進路ロードマップは価値がある | サンプルを提示し、自分の子ども版が欲しいか聞く | 5名中3名以上が欲しい |
| 進路相談は有料化できる | 単発・月次の価格を提示する | 5名中2名以上が有料相談を検討 |
| 集団指導は個別より利益率の高い主力になり得る | 少人数クラス案を提示する | 5名中2名以上が月3万から5万円を検討 |
| コミュニティは中間商品になり得る | 月額会員の中身を提示する | 5名中2名以上が月3,000円前後を検討 |
| 中国語対応はターゲット特化価値になり得る | 後半で在日中国人家庭に置き換えて評価してもらう | 5名中3名以上が価値が上がると回答 |

## 検証しないこと

以下は今回の日本人テストでは判断しない。

- 中国語で説明されたときの理解度・安心感
- 在日中国人家庭の実際の支払意欲
- WeChatやRED NOTEでの集客可能性
- 中国人保護者が求める細かい表現・文化的配慮

これらは、別途在日中国人家庭へのヒアリングで検証する。

## 成功条件

今回のテストで以下が確認できれば、次に進む価値がある。

- 進路ロードマップと進路相談への関心が明確に出る
- 商品ラインの違いが理解される
- 集団指導またはコミュニティのどちらかに支払意欲が出る
- 「言語差がある家庭なら価値が上がる」という第三者視点の納得がある

## 失敗条件

以下の場合は、資料・商品設計を見直す。

- 進路ロードマップの使い道が伝わらない
- 4商品が多すぎて理解されない
- 価格提示で一気に関心が下がる
- 中国人家庭向けである理由が第三者にも伝わらない

## 当日の説明方針

最初から中国語対応を強調しすぎない。前半は日本人家庭として評価してもらい、後半で「もし日本語で受験情報を取りにくい家庭だったらどうか」と置き換えてもらう。

この順番にすることで、サービスの普遍的価値とターゲット特化価値を分けて見る。
