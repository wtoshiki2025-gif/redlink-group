# オンライン授業 不安・必要機能 確認シート

## 目的

オンライン個別指導・オンライン集団指導について、保護者が何に不安を感じるか、どの機能があれば申し込みやすくなるかを確認する。

このシートは、オンライン授業の説明前または説明後に見せる。最初に自由回答を聞き、その後で下記の選択肢を見せる。

## 事前確認

オンライン授業への反応は、現在の学習環境によって分けて見る。

| 確認項目 | 内容 |
| --- | --- |
| 対面の塾 | 進学塾、補習塾、個別指導塾などに通っているか |
| 家庭教師 | 対面またはオンラインの家庭教師を雇っているか |
| オンライン授業経験 | オンライン塾、オンライン個別指導、映像授業を使ったことがあるか |

## 確認したい不安

該当するものを選んでもらい、特に強いものを3つまで選んでもらう。

| 不安 | 内容 |
| --- | --- |
| 集中できるか | 子どもが画面越しに集中し続けられるか不安 |
| 質問できるか | わからないところをその場で質問できるか不安 |
| 理解度が見えるか | 先生が子どもの理解度を把握できるか不安 |
| 授業品質 | 先生の教え方や受験指導力に差がありそう |
| 相性 | 先生と子どもの相性が合わない可能性 |
| 学習管理 | 授業外の宿題・復習まで管理されるか不安 |
| 保護者の見えにくさ | 授業中の様子や進捗が親から見えにくい |
| 競争感 | 対面塾のような緊張感や競争感が弱そう |
| 料金納得 | 対面より安くないと納得しづらい |
| 機材・環境 | 通信環境、カメラ、音声、教材共有が不安 |

## 欲しい機能・サポート

「これがあれば不安が下がるもの」を選んでもらい、特に価値が高いものを3つまで選んでもらう。

| 機能・サポート | 内容 |
| --- | --- |
| 体験授業 | 本申込前に先生・授業形式を試せる |
| 先生プロフィール | 経歴、受験経験、得意科目、指導実績が見える |
| 先生変更制度 | 相性が合わない場合に変更できる |
| 授業後レポート | 学習内容、理解度、宿題、次回までの課題が届く |
| 確認テスト | 授業後に理解度を測る小テストがある |
| 宿題管理 | 宿題の提出・未提出・復習状況を管理する |
| 録画・アーカイブ | 授業を後から見返せる |
| 保護者面談 | 月1回など、親が先生や運営と話せる |
| 学習ロードマップ | 志望校から逆算した学習計画がある |
| 進捗ダッシュボード | 成績、宿題、テスト結果を一覧で見られる |
| 授業中の見守り | 運営が授業品質や参加状況を確認する |
| 欠席振替 | 欠席時に振替や録画視聴ができる |

## 深掘り質問

- オンラインによる授業をどう思いますか。
- 対面授業と比べて、オンラインの良い点・悪い点は何ですか。
- 一番不安が強いものはどれですか。なぜですか。
- その不安があると、申し込みをやめる理由になりますか。
- どの機能があれば「オンラインでもよい」と思えますか。
- 個別指導と集団指導で、不安は変わりますか。
- 対面塾と同じ価格でも納得できますか。安くないと難しいですか。
- 保護者向けレポートは、どの頻度・内容なら価値がありますか。
- 在日中国人家庭の場合、中国語での保護者説明やレポートがあると不安は下がると思いますか。
