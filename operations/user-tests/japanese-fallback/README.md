# 日本人モニター対応版 初期ユーザーテスト

作成日: 2026-05-12

## 位置づけ

本来は在日中国人家庭を対象に初期ユーザーテストを行う予定だったが、今回の調査会社手配では中国人モニターを確保できなかった。そのため、日本人保護者を代替モニターとして使い、サービスのうち「日本の受験支援として普遍的に価値がある部分」を検証する。

このテストでは、在日中国人家庭における中国語対応の本当の支払意欲は直接検証できない。したがって、結果は「本命ターゲットの需要証明」ではなく、商品設計・説明資料・価格帯・ヒアリング設計の事前検証として扱う。

## 検証できること

- 進路ロードマップが保護者にとって有用に見えるか
- 受験制度、塾選び、学習計画の相談ニーズがあるか
- オンライン進路相談、個別指導、集団指導、コミュニティの違いが理解されるか
- 各商品の価格帯が教育熱心な家庭にとって現実的か
- 日本人保護者から見て、在日中国人家庭向けの切り口に説得力があるか

## 検証できないこと

- 中国語対応に対する実際の支払意欲
- 在日中国人家庭の情報収集行動
- RED NOTE、WeChat経由の集客可能性
- 中国語での説明資料・面談品質
- 在日中国人コミュニティ内での紹介発生可能性

## 判断基準

日本人モニター5名で実施する場合、以下を目安にする。

| 項目 | Go寄りの反応 | 注意が必要な反応 |
| --- | --- | --- |
| 進路ロードマップ | 3名以上が「自分の子ども版が欲しい」 | 具体的な使い道が出ない |
| 進路相談 | 2名以上が有料相談を検討 | 無料なら聞く、で止まる |
| 集団指導 | 2名以上が月3万から5万円を検討 | 個別との差が伝わらない |
| コミュニティ | 2名以上が月3,000円前後を検討 | チャットグループ程度に見られる |
| 中国人家庭向け説得力 | 3名以上が「言語差がある家庭なら価値が上がる」と回答 | 日本人向けとの差別化が弱い |

## 重要な注意

今回の結果で「在日中国人家庭に売れる」と結論づけない。日本人モニターで強かった反応は普遍的価値として扱い、中国語対応に関する反応はあくまで第三者視点の仮説補強として扱う。
