# 初期ユーザーテスト 当日提示スライド

目的: 保護者から、進路支援・オンライン授業・価格・在日中国人家庭向け価値への率直な反応を取る。

使い方:

- 1セクションを1スライドとして使う。
- 各スライドの「質問」は、その場で参加者に聞く。
- 回答は `recording-sheet.md` に記録する。
- 参加者には正解を探してもらわず、わかりにくい点・不要な点・高い点を率直に話してもらう。

---

## スライド1: 今日の前提

本日は、在日中国人家庭向けのオンライン進学支援サービスについてご意見を伺います。

今回は日本人保護者の方に、以下の2つの視点でお聞きします。

1. ご自身の家庭だったらどう感じるか
2. 日本語で受験情報を取りにくい家庭だったらどう感じるか

今日は良い評価をいただくことが目的ではありません。
わかりにくい点、いらないと思う点、高いと感じる点も率直に教えてください。

**質問**

- お子さまの学年を教えてください。
- 現在、塾・家庭教師・通信教育などを利用していますか。
- 中学受験、高校受験、大学受験への関心はありますか。

## スライド2: 保護者の受験・学習管理の悩み

保護者が抱えやすい悩みとして、以下を想定しています。

**質問**

- 最近、進路や塾選びで迷ったことはありますか。
