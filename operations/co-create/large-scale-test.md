## ターゲット来場者の引き込み方

**課題:** SSTフェスの来場者は主に日本人。在日中国人の保護者は少数。

**対策:**
- パネルに中国語を入れる → 中国語を見た中国人が自然に立ち寄る
- スタッフ（和久井）が中国語でカジュアルに声かけできる状態にする
- 日本人が来た場合は「中国人の知り合いがいれば紹介してほしい」とブランド認知につなげる
