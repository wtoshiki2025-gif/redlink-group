## 集客戦略

**メイン：日本人家庭（当日飛び込み）**

来場者は主に日本人のため、日本人保護者への相談会をメインとして設計する。Phase 2（日本人家庭への展開）の先行検証として位置づける。

- テント外の立て看板・声かけは日本語メイン
- テント外で声かけ → 興味があれば着席
- 相談中は「次の空き時間に来てください」と伝えて一旦帰す

**サブ：在日中国人保護者（Rednote / WeChat 経由・事前告知済み）**

Rednote・WeChatで「医学部生による受験・進路相談」として事前告知し、連絡が来た方のみ当日ブースへ案内する。

- 中国語での対応可
- ロードマップ・書画カメラデモを中国語で説明
- 連絡手段はWeChatを優先

## 相談セッション概要（1組あたり15〜20分）

言語は日本語基本。中国人保護者が来た場合は中国語でも対応可。

1. **自己紹介** — 鉄緑会出身・医学部生であることを伝える
2. **ヒアリング** — 子どもの学年・志望校・現在の塾状況を聞く
3. **進路・受験相談** — 今の悩みに対して、次に何を決めるべきか整理する
4. **ロードマップ提示** — サンプルを見せながら受験までの道筋を説明
5. **書画カメラ実演** — オンライン授業で手元を映し、解き方・記述・添削をどう見るか実演する
6. **次アクション提案** — 無料面談、体験授業、継続授業のどれに関心があるか確認する

即決でなくてもOK。**連絡先交換だけは必ず完了させる。**

## 相談テーマ一覧

### 共通（日本人・中国人両方）

| テーマ | 具体的な切り口 |
|--------|--------------|
| 今の学力と志望校のギャップ | 模試・偏差値から何が足りないかを整理 |
| 塾・家庭教師の選び方 | 今の環境で十分か、何が足りないか |
| 科目別の優先順位 | 何から手をつけるべきか |
| 受験スケジュールの組み方 | 逆算して各時期に何をすべきか |
| 医学部・難関大学への現実的な道筋 | 自分の受験経験を交えて |

### 日本人保護者向けに重点を置くテーマ

| テーマ | 具体的な切り口 |
|--------|--------------|
| 中学受験 vs 高校受験の選択 | それぞれのメリット・デメリットを整理 |
| 推薦・内申 vs 一般入試 | どちらを軸にすべきか |
| 子どもがやる気を出さない | 医学部生として経験した転換点の話 |

### 中国人保護者向けに重点を置くテーマ

| テーマ | 具体的な切り口 |
|--------|--------------|
| 日本の受験システムの全体像 | 内申・推薦・一般の仕組みを説明 |
| 日本語力と受験科目の関係 | 国語をどう扱うか |
| 帰国子女入試・外国人枠の有無 | 学校によって選択肢が異なる点を案内 |
| 日中の教育方針の違い | 日本の受験に合わせた切り替え方 |

## アプローチの違い（日本人 vs 中国人）

| | 日本人（飛び込み） | 中国人（事前来訪） |
|---|---|---|
| 入り口 | 声かけ → 自己紹介 → ヒアリング | 確認のひと言だけでヒアリングへ |
| 重点説明 | 書画カメラの新しさ・手元指導のイメージ | 日本の受験システム＋中国語対応できる安心感 |
| 連絡手段 | LINE | WeChat 優先 |
| 言語 | 日本語 | 中国語でも可 |
