## 1. 運用の基本原則

### 中華圏市場 vs 日本市場の違い

| 観点 | 日本 | 中華圏 |
|------|------|--------|
| 信頼の源泉 | ブランドの世界観・歴史・一貫した美意識 | 実際に使っている人のリアルな声 |
| 重視される情報 | ブランド公式の発信 | UGC（ユーザー生成コンテンツ）、KOCの体験談 |
| 購買決定 | ブランド＝信頼 | 口コミ＝信頼 |

### RED NOTE運用の核心

- **ユーザー観点で伝えること**が最も重要
- 広告っぽさが出過ぎるとNG — リアルな声で書くと伸びやすい
- 硬すぎる文章は避ける
- 翻訳色が強すぎると読まれない — 機械的な翻訳ではなく中国語ネイティブの自然な表現が必要
- ユーザーのためになる情報を記事にする

### 中華圏トレンドの活用

- 中国のネットトレンドに乗るべき（特に中華圏特有のイベント）
- RED NOTEがおすすめしている流行ハッシュタグを活用する
- **毎月のトレンド・イベントカレンダーを整理しておくと強い**
- 主要イベント: 春節、618プロモーション、ダブルイレブン（11.11）など

## 2. アカウント管理

### 端末・ログインルール

- **1台の端末に1アカウント**が基本。頻繁な切り替えはマーケティングアカウントと認識されるリスクあり
- 2台のスマホで同じアカウントへの同時ログインは不可（片方が自動ログアウト）
- 1つのWiFiに接続できるアカウントは**最大3つ**まで。公共WiFiは非推奨

### アカウント育成期間

- **個人アカウント**: 約1週間
- **企業アカウント**: 約3日間
- 育成期間中は通常のユーザーのようにアプリを使用（閲覧・交流）

### アカウント認証

- **身分証明書1つ** → 認証可能アカウント: 1つ
- **営業許可証1つ** → 認証可能企業アカウント: 最大3つ

### 個人アカウント vs 企業アカウント

| 項目 | 個人アカウント | 企業アカウント |
|------|--------------|--------------|
| 対象 | 一般ブロガー | 営業許可証を持つ事業者 |
| 外部誘導 | 制限あり | 比較的可能 |
| 育成期間 | 約1週間 | 約3日 |
| 機能 | 基本機能 | 広告契約、ショップ開設、薯条配信 |

### 専門アカウント（专业号）の認証メリット

- ブランド広告の契約
- ショップ（薯店）の開設（フォロワー0でも可。ただしカテゴリにより資格が必要）
- 広告配信（薯条）の実施

### プロフィール設定

- 変更は**週3回まで**可能（回数自体は評価に大きな影響なし）
- 違反プロフィールの頻繁な変更は評価低下の原因
- 連絡先（WeChat、電話番号）の記載は厳禁
- メールアドレスは比較的安全だが、自動返信でWeChatを伝える設定が推奨
- QRコードの使用は**絶対NG**（アイコン、投稿、画像すべてで違反）

### アカウント評価への影響

- アカウント評価が低い → おすすめ・検索表示の確率が低下
- **投稿の大量削除は非推奨** → 評価低下・異常アカウント判定のリスク
- **データ偽造**（いいね・フォロワー購入）→ アルゴリズムが検出し削除、違反記録が残る
- 複数アカウントでの同一顔写真使用は、内容が重複せず違反がなければOK

## 3. コンテンツ戦略

### 投稿形式

- **静止画投稿**: 化粧品などに適する
- **動画投稿**: 視覚的にわかりやすい（2〜5分が最適）
- **ポスター風投稿**
- 写真を動画テンプレートで変換して投稿も有効（カバー画像・音楽・テキスト配置に工夫が必要）
- 記事 vs 動画: 質が高ければどちらでもOK。新規ユーザーは記事から始めるのが良い
- 15分以上の動画投稿には動画アカウント申請が必要（フォロワー2,500人以上、1分以上の投稿歴など）

### サムネイル・TOP画像

- 推奨サイズ: **3:4（縦長）** が最も目を引く
- 他に1:1、4:3も選択可能

### コンテンツの「垂直性」

- 特定ジャンルに一貫したコンテンツを投稿すること
- 初期はジャンルを絞り、後から拡張するのが推奨
- 様々なジャンルを一度に投稿 → フォロー率が大きく低下
- 異なるターゲット向けのコンテンツは**別アカウント**を作成すべき

### 投稿頻度

- **理想**: 週3〜5回
- 毎日更新も質が保てれば効果的
- 最低でも週1回以上

### 投稿時間帯（中国時間）

- 朝: 6:00〜8:00
- 昼: 11:00〜13:00
- 夜: 17:00〜21:00
- ※日本と中国の時差（1時間）を考慮すること
- 曜日: 水曜日が最もトラフィック良好、金曜日が最少という結果あり（ジャンルにより異なる）

### ハッシュタグ

- 適切な数のハッシュタグを選ぶ（大量に羅列するだけでは効果なし）
- コンテンツに関連するトレンドワード（熱捜）を使用する
- 無関係なトレンドワードの乱用は効果なし

### 企業アカウントの効果的運用

- プロフィール・TOPページで「何を提供しているか」「強み」を明確に
- 投稿に一貫性を持たせる → イメージ向上
- モデル起用 → 使用シーンがわかるもの
- 百均インテリアを活用した写真制作も有効
- インフルエンサー投稿の二次利用 → **交渉段階で二次利用可否を確定**
- プレゼント抽選キャンペーンでフォロワーと交流（ルールは都度確認）
- 「ソフト広告」として商品レビューや使い方紹介が共感を得やすい

### アクティブ運用

- 自社商品を投稿しているユーザーにリアクションする
- ネガティブ投稿にも丁寧に対応 → アンチからファンへ

## 4. 禁止事項・注意点

### 絶対NG

- **外部リンクの送信** → 重大な違反
- **QRコードの表示**（アイコン・投稿・画像すべて）
- **WeChat ID・電話番号のプロフィール記載**
- **「瞬間」機能へのWeChat情報記載**
- **データ偽造**（いいね・フォロワー購入）
- **手術のBefore/After画像**

### 敏感ワード・制限対象

- RED NOTEの敏感ワードリストに注意
- 極端な表現は避ける
- **政治・宗教**に関する内容は要注意
- 注意が必要なカテゴリ: 美容医療、コスメ、ダイエット、サプリメント、不動産
- 占星術は迷信と見なされ制限される可能性あり
- 健康・医療関連の投稿は「効果未確認」と判断されると制限

### コメント欄の注意

- アプリ外の連絡先・URL誘導に関連する言葉 → 自動削除される
- 他プラットフォーム名（淘宝タオバオなど）の直接言及 → 違反の可能性
  - 工夫例: 淘宝を「桃」の絵文字で表現

### 広告投稿の注意

- 公式に報告していないPR投稿 → 原則プラットフォーム経由が必要（手数料あり・資格必要）
- 広告が目立たない場合は問題ないこともある（例: 観光地推薦はPRと認識されにくい）
- 広告違反でアカウント制限された場合 → 投稿を削除し1〜2ヶ月は広告を控える

## 5. 外部誘導の方法と注意

### 基本ルール

- DMでのリンク送信は違反
- コメント欄でのリンク送信も違反リスクあり
- RED NOTE IDをWeChat IDと同じにする誘導は非推奨

### 比較的安全な方法

- **メールアドレスをプロフィールに記載** → 自動返信でWeChat情報を提供
- コメントで「DMで返信しました」と記載し、DMでやりとり（URLは送信しない）
- ユーザーに検索キーワードを教え、Google/Baiduで自ら検索してもらう
- **ファングループを作成**し、そこにユーザーを誘導（最も安全）

### WeChat誘導の暗号

- 「微信」という言葉はリスク高い
- 代替語: 「加V」「VX」など
- 画像を使って伝える方法もあり
- **同じ表現を複数人に繰り返さないこと**

### 違反した場合

- 誘導違反の通知を受けたら**最低1週間は誘導行為を停止**
- アカウント育成に専念する

## 6. SEO対策

### なぜSEO対策が必要か

- RED NOTEユーザーの**80%以上**が検索を利用
- 約**90%**のユーザーが検索結果が購買決定に影響すると回答
- 中国ではBaiduよりSNS検索が主流に移行（Googleアクセス不可、Baidu信頼性問題）
- **ロングテール流量**: 推薦が終わっても検索経由で継続的にトラフィック獲得
- インバウンド向けは「バズる」瞬間風速よりSEO・ロングテールを重視すべき

### 検索の仕組み（逆じょうごモデル）

1. ユーザーが広義の検索（例:「冬用クリーム」）
2. 検索結果から興味のあるブランド・製品を絞り込み
3. 評価を総合的に考慮して購入決定

### キーワード選定

**① 製品の種類から選定**
- 例:「水彩ペン」「クリーム」「日焼け止め」
- 業界ワードは競争が激しいため、さらに細分化する

**② 4つのワードカテゴリで細分化**

| カテゴリ | 例 |
|---------|-----|
| 機能ワード | 美容、修復、消毒、リフトアップ |
| 場面ワード | 徹夜、秋冬、クリスマス、旅行時の撮影 |
| ユーザー群ワード | 初心者、会社員、学生 |
| 課題解決ワード | 無料で便利なアプリ、低価格で実用的、コスパ抜群 |

### キーワードの活用方法

- **タイトル、ハッシュタグ、表紙**にキーワードを自然に配置
- 内容の関連性を高める（アカウントと話題の関連性がランクに影響）
- 長期的・継続的に高品質コンテンツを投稿する

### 収録（インデックス）

- 高品質アカウントの高品質投稿が優先的に収録
- 収録されにくい投稿:
  - ルール違反がある
  - 敏感ワードを含む
  - アカウントが制限状態
  - 内容の質が低い
- 確認方法: 「灰豚データ」等のツールで確認
- 収録されない場合 → 検索経由のトラフィックが得られず露出が大幅減少
- 古い投稿の修正で収録される可能性はあるが保証なし
- 全投稿が検索非表示 → 重大な違反の可能性。アカウント再育成or新規作成を検討

## 7. 広告・プロモーション

### アプリ内広告の種類

| 種類 | 説明 |
|------|------|
| リスティング広告 | ユーザーの検索時に自社広告を上位表示 |
| フィード広告 | フォロワーのフィード内に自社コンテンツを混ぜて表示 |
| 薯条 | 投稿の露出を増やす広告商品 |

### 薯条の活用

- いいね数、お気に入り数、閲覧数のいずれかに投資可能
- 最初は**少額でテスト**しながら調整
- 時間帯はターゲット層により異なる（例: 年齢層高め → 朝6時から効果的な場合あり）
- 期間: **12〜24時間**が一般的
- **外部誘導目的の投稿には薯条を買わない**（制限リスク）

### 商品サンプリング

- **発売3ヶ月以内**の商品はサンプリングが可能
- 顧客セグメントを絞って広告配信が可能

### ローカル店舗のプロモーション

- アカウント・投稿に**位置情報を設定**（中国以外の位置情報も設定可能）
- 投稿に地域関連キーワードを含める
- 企業アカウントでは店舗住所の関連付けが可能

### 注意事項

- フォロワー数が多い ≠ 来店効果が良い（コンバージョンはコンテンツの質次第）
- バズった投稿 ≠ 購入・問い合わせにつながる（正確なプロモーションが重要）
- 複数ブランド比較投稿 → 通報リスクあり。公平中立に（同一ブランド内比較は比較的安全）

## 8. バズ投稿の活用法

### バズ判定の目安

- 投稿後**1〜3日**以内にバズることが多い
- 投稿後**3時間以内**の閲覧・いいねの急増が指標
- 各投稿のアルゴリズムは独立 → 1つバズっても他の投稿には影響しない

### バズ投稿自体の最大活用

1. **薯条への投資** — いいね数を増やしクリック率を向上（表紙・タイトルは変更しない）
2. **コメント欄での情報補足** — 二次編集よりコメント欄での補足が安全
3. **コメント欄の管理**
   - 重要コメントを固定（購入場所、懸念への回答など）
   - 購買ムードの醸成（リンク要求、価格質問、購入報告のコメント）
   - 画像コメントでリアルさを強調
4. **検索結果ページの最適化**
   - 関連性の低いコンテンツの表示を防ぐ
   - ネガティブコンテンツの対策（ポジティブコメント、ブランドスペース購入）

### バズ投稿の二次利用

| 方法 | 詳細 |
|------|------|
| 繰り返し利用 | 非PR→PR化、インフルエンサーから権利取得し自社再投稿、自社コンテンツの継続利用 |
| 模倣・復刻 | バズ直後の短期間に同様の形式で作成（模倣者が増える前に） |
| 小規模リニューアル | バズ要因を特定し新要素を追加 |

### モジュール思考に基づく活用

- **表紙の再利用**: 他投稿の表紙をバズ投稿の表紙に差し替え
- **スクリーンショットの活用**: バズ投稿のスクショを表紙に利用、データ含むスクショで信頼性強化
- **形式変換**: テキスト→動画、動画→テキスト
- **バズ要因に基づく展開**: 製品の異なるモデルや詳細特性にフォーカス

## 9. 炎上・クレーム対応

### 対応が必要な範囲

- ブランド名・製品名で検索した際に**検索結果上位にネガティブ投稿が表示される場合**

### 分析のポイント（まず冷静に）

- **発信者は誰か？**（顧客、競合企業など）
- **何に対する不満か？**（ブランド、製品、サービス）
- **目的は何か？**（感情発散、解決策要求、誹謗中傷）

### 顧客からのクレーム対応

| 目的 | 対応方法 |
|------|---------|
| 感情の発散 | ポジティブコメントで誘導、使用方法や個別ケースとして説明 |
| 解決策の要求 | 公式アカウントでまず謝罪→状況ヒアリング→対応期限伝達→解決。コメント欄で状況説明しポジティブコメント固定 |
| 悪意ある投稿 | ①通報 ②第三者技術機関に依頼（市場価格約20万円〜） |

### 競合企業からのネガティブ投稿対応

1. 通報する
2. ポジティブコメントで誘導（ネガコメ多数の場合は非効率）
3. 第三者機関に依頼（検索順位操作）
4. ポジティブ投稿で対抗
5. 声明を発表

### 対応方法まとめ

1. ポジティブコメントで誘導
2. 正面的な対応
3. 通報
4. 第三者機関による検索順位操作
5. ポジティブ投稿での対抗
6. 声明の発表

### 心構え

> 「軽微なクレームがあることは正常、クレームが全くない方が異常」

- 炎上原因を分析し、ブランド・製品・サービスの改善につなげる
- 普段の投稿で過度なPRを避け、誠実さを重視する

## 10. インフルエンサー分類と用語

### インフルエンサーランク（灰豚データ基準）

| ランク | フォロワー数 |
|--------|------------|
| トップインフルエンサー | 50万人以上 |
| 中堅インフルエンサー | 5万人〜50万人 |
| 初級インフルエンサー | 5,000人〜5万人 |
| 一般ユーザー | 300人〜5,000人 |

### KOL（Key Opinion Leader）

- 特定分野で発言力と影響力を持ち、多数のフォロワーを持つ個人
- 日本語の「インフルエンサー」と同義
- 強力なコミュニティ基盤とマーケティング価値
- リスク: キャラクター崩壊によるブランドへの広報危機

### KOC（Key Opinion Consumer）

- 「重要意見消費者」— 自分の体験を共有し友人やフォロワーの購買に影響を与える
- KOLより少ないフォロワーだが「信頼性」「リアルさ」が強み
- 消費者の立場からの実際の感想 → ターゲットに特化、コスパが高い
- **RED小紅書ではKOCのパワーが特に強い**

### KOS（Key Opinion Sales）

- 「重要意見販売者」— 専門的販売能力と業界知識を持つコンテンツクリエイター
- KOLとKOCの長所を融合
- 「販売ガイド」として認められている（KOLは広告で批判されやすいがKOSは販売が認められる）
- 適した領域: 医薬、車、美容、健康食品など高度な専門知識が必要な分野

### KOF（Key Opinion Followers）

- ブランドの忠実なファン
- 新商品をすぐ購入し、友人にも推薦する存在

### KOB（Key Opinion Buyer/Business）

- 販売業者や店舗
- 「企業アカウント」「自社アカウント」に近い概念

## 11. アルゴリズム変更（2024年秋）

### 背景

- 2024年9月末〜11月初め: 閲覧数が急激に減少（多くのアカウントで投稿が数百回しか閲覧されない状況）
- RED小紅書がバイヤーエコマースモードを試験導入した初年度

### トラフィック減少の原因

#### ① ダブルイレブンの影響

- 約2ヶ月前からPR活動開始 → セール1ヶ月前に広告投資がピーク（「マーケティングの蓄水期間」）
- トラフィックが商業広告に偏り → オーガニックトラフィックが大幅減少
- **同様の現象が起こる時期**: 春節前の「年貨節」、6月の「618プロモーション」（イベント3〜4週間前から）

#### ② 新ルールによるアルゴリズム変更

**（1）低品質アカウントの取り締まり**
- 「投稿ロボット」のようなアカウントが厳しく取り締まり対象
- 他人のコンテンツ転載や過去投稿のコピー → 「非誠実な創作」

**（2）低品質・重複コンテンツの制限**
- 同じ内容の微修正での繰り返し投稿 → ペナルティ
- 投稿制限、プロモーション制限、ライブ配信のトラフィック制限

**（3）アカウントスコアリング・信頼スコアの変更**
- 新規アカウントは**7日間の観察期間**（閲覧数が非常に低い）
- **日次活躍度** = 1日の平均使用時間 ÷ 投稿数（短時間使用で大量投稿 → ロボット判定）

### 新ルール下での対策

1. **専門運営者の配置**
   - コンテンツのオリジナル制作能力
   - 中国ネットトレンドの判断力
   - 理想的には運営者自身がインフルエンサー

2. **コンテンツ・投稿戦略の調整**
   - 質の高いコンテンツを定期投稿
   - 積極的にコメント運営・ユーザーとコミュニケーション
   - トレンドや人気タグの活用
   - 機械的翻訳ではなく中国トレンドを取り入れた新コンテンツ作成

## 12. データ分析

### 主要指標

| 指標 | 説明 |
|------|------|
| 小眼睛（目の数） | 投稿のクリック数（2024年10月まではインプレッション数だった説も） |
| クリック率 | 10%以上 = 非常に良い、5〜10% = 質の良い投稿 |
| いいね・お気に入り | コンテンツの魅力を反映 |

### 新規アカウントの目安

- 閲覧数100〜200は正常（アカウント新しく評価低いため）
- 長期運営で閲覧数が低い場合 → コンテンツの問題、違反、低品質、頻繁なプロモーションが原因の可能性

### フォロワー・いいね数の減少原因

- システムによる不正データ（偽フォロワー・偽いいね）の削除
- ユーザー自身のフォロー解除・お気に入り取り消し

### データ確認ツール

- **クリエイターセンター**: 過去7日間・30日間のフォロワー状況と投稿データ
- **灰豚データ**: 過去データの確認、投稿の収録確認、競合アカウント検索

## 業種別Tips

### 化粧品・美容

- 静止画投稿が適している
- 美容医療は敏感カテゴリ（手術Before/Afterは絶対NG）

### ローカル店舗（飲食等）

- 位置情報の設定が必須
- 店主の日常共有で親近感を醸成
- 企業アカウントでターゲットを絞ったプロモーション

### 健康・医療関連

- 誇張表現を避ける
- 具体的な製品名の使用を控える
- リアルな人物として「健康的な生活」を共有するアプローチが効果的

### 書籍・映画レビュー

- 映画レビュー: フォロワー増加速度が速い
- 書籍レビュー: 比較的遅い
- 関連商品販売や広告で収益化

### ワイン・グルメ

- 初期は専門知識・科学的内容で信頼構築
- 産地紹介、初心者ガイド、レビューガイド、ペアリングテクニックなど
